Study Material:

Artificial Intelligence is a field of computer science focused on creating systems capable of performing tasks that normally require human intelligence. Machine Learning is a subset of AI that enables systems to learn patterns from data without being explicitly programmed. Deep Learning is a subset of Machine Learning that uses neural networks with multiple layers.

A neural network consists of input layers, hidden layers, and output layers. Training a neural network requires data, a loss function, and an optimization algorithm. Gradient Descent is one of the most commonly used optimization techniques.

Supervised learning uses labeled datasets. Unsupervised learning uses unlabeled datasets. Reinforcement learning trains agents using rewards and penalties.

Large Language Models (LLMs) are trained on massive text corpora. Transformer architectures introduced self-attention mechanisms that significantly improved language understanding. Attention mechanisms allow models to focus on relevant parts of the input sequence.

Retrieval-Augmented Generation (RAG) combines information retrieval with language generation. RAG systems typically consist of document ingestion, chunking, embedding generation, vector storage, retrieval, and answer generation.

Chunking is the process of splitting large documents into smaller sections. Poor chunking strategies can reduce retrieval quality. Semantic chunking generally performs better than fixed-size chunking for complex documents.

Embeddings are numerical vector representations of text. Similar concepts produce vectors that are closer together in vector space. Cosine similarity is frequently used to compare embeddings.

Vector databases store embeddings for efficient retrieval. Popular vector databases include ChromaDB, Qdrant, Pinecone, Weaviate, and Milvus.

ChromaDB is lightweight and developer-friendly. Qdrant provides strong filtering capabilities. Pinecone is a managed cloud solution. Milvus is designed for large-scale deployments.

A common RAG pipeline consists of:

1. Document Loading
2. Text Cleaning
3. Chunking
4. Embedding Generation
5. Vector Storage
6. Similarity Search
7. Context Injection
8. LLM Response Generation

Agentic AI systems can perform actions autonomously. Agents typically use planning, memory, tools, and reasoning.

Short-term memory stores information relevant to the current conversation. Long-term memory stores persistent information across interactions.

LangGraph is a framework for building stateful AI agents. It represents workflows as graphs. Nodes perform tasks while edges determine transitions between states.

CrewAI focuses on collaborative multi-agent workflows. AutoGen emphasizes communication between multiple autonomous agents.

A decision made by an agent should be preserved in memory because future actions may depend on that decision.

Constraints must never be removed from memory. If an agent receives the instruction "Do not use cloud APIs," the constraint must persist even after context compression.

Requirements specify what must be included in a solution. Constraints specify what must not be violated.

Example:

Requirement:
Use LangGraph.

Constraint:
Do not use OpenAI APIs.

A correct implementation satisfies both conditions simultaneously.

Databases can be categorized as relational or non-relational. PostgreSQL is a relational database. MongoDB is a document-oriented database. Redis is an in-memory key-value store.

ACID stands for Atomicity, Consistency, Isolation, and Durability. ACID properties are critical for transactional systems.

CAP theorem states that distributed systems can guarantee only two of the following three properties simultaneously:

1. Consistency
2. Availability
3. Partition Tolerance

Cloud computing is commonly divided into IaaS, PaaS, and SaaS.

IaaS provides infrastructure resources.
PaaS provides application development platforms.
SaaS provides complete software solutions.

Docker enables application containerization. Containers package applications and dependencies together.

Kubernetes orchestrates containers across clusters. Pods are the smallest deployable units in Kubernetes.

REST APIs use HTTP methods such as GET, POST, PUT, and DELETE.

GET retrieves data.
POST creates data.
PUT updates data.
DELETE removes data.

Authentication verifies identity.
Authorization determines permissions.

OAuth 2.0 is a widely used authorization framework.

Cybersecurity principles include:

* Least Privilege
* Defense in Depth
* Zero Trust

Zero Trust assumes that no user or system should be trusted by default.

Software engineering best practices include:

* Version Control
* Automated Testing
* Continuous Integration
* Code Reviews
* Documentation

Git is the most widely used version control system.

Unit tests validate individual components.
Integration tests validate interactions between components.
End-to-end tests validate complete workflows.

Performance optimization should only occur after identifying bottlenecks through measurement.

Premature optimization can increase complexity without providing meaningful benefits.
